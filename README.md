# Strategy Map: Van Data naar Waarde

Een interactieve webapplicatie die een strategische kaart visualiseert op basis van het Six Capitals-model, ontworpen voor gebruik in het bedrijfskundig adviestraject.

## Doel

Deze applicatie helpt studenten en adviseurs om de **oorzaak-gevolgrelaties** binnen een organisatiestrategie inzichtelijk te maken. Via een visuele strategy map worden de verbanden tussen verschillende kapitaalvormen en de uiteindelijke waardecreatie voor stakeholders weergegeven.

## Functies

- **Strategy Map Visualisatie**: Interactieve weergave van strategische doelen en hun onderlinge verbanden, geordend per perspectief (Impact, Klant, Intern, Leren & Groei).
- **Six Capitals Model**: De kaart is opgebouwd rond zes kapitaalvormen: intellectueel, sociaal, menselijk, geproduceerd, financieel en natuurlijk kapitaal.
- **Holistische Waardecreatie**: Toont hoe interne processen en capaciteiten leiden tot klantwaarde en uiteindelijk tot brede maatschappelijke impact.
- **Uitlegmodus**: Aparte tab met toelichting op de gebruikte methodiek en het theoretisch kader.
- **Enkelvoudige HTML-pagina**: Geen installatie vereist, direct bruikbaar via de browser of GitHub Pages.

## Gebruik

De applicatie is bedoeld als onderwijsmateriaal voor HBO Bedrijfskunde, met name voor modules rondom strategie, adviesvaardigheden en waardecreatie. Studenten kunnen de kaart gebruiken als template of referentie voor het opstellen van een eigen strategy map voor hun opdrachtgever.
